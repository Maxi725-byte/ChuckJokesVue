# ChuckJokesVue

## Descripción
Proyecto desarrollado con Vue.js para mostrar una lista de chistes de Chuck Norris utilizando la directiva v-for.

## Requisitos
- Servidor local (XAMPP/Laragon)
- Vue.js via CDN

## Instrucciones de uso
Colocar los archivos en la carpeta de servidor local y acceder vía localhost/ChuckJokesVue.
